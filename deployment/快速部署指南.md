# ⚡ Carbon ERP/MES 快速部署指南

**5 步完成生产环境部署**

---

## 📋 前置要求

- ✅ 阿里云 ECS 服务器（4核8GB, Ubuntu 22.04）
- ✅ 域名（如 yourcompany.com）
- ✅ SSH 访问权限

---

## 🚀 5 步部署

### 第 1 步：安装依赖（10 分钟）

```bash
# 连接服务器
ssh root@your-server-ip

# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 安装 Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# 安装 pnpm 和 PM2
npm install -g pnpm pm2

# 安装 Nginx
sudo apt install -y nginx
```

### 第 2 步：配置域名（5 分钟）

在域名提供商处添加 DNS 解析：

```
A记录: erp.yourcompany.com  → 服务器IP
A记录: mes.yourcompany.com  → 服务器IP
A记录: api.yourcompany.com  → 服务器IP
```

### 第 3 步：部署 Supabase（15 分钟）

```bash
# 克隆 Supabase
git clone --depth 1 https://github.com/supabase/supabase
cd supabase/docker

# 配置环境变量
cp .env.example .env
nano .env

# 修改以下配置:
# POSTGRES_PASSWORD=your-super-secret-password
# JWT_SECRET=your-super-secret-jwt-token-32-chars
# SITE_URL=https://api.yourcompany.com

# 启动 Supabase
docker compose up -d

# 等待启动完成（约 2 分钟）
docker compose ps
```

### 第 4 步：部署 Carbon（20 分钟）

```bash
# 克隆项目
cd ~
git clone your-carbon-repo.git carbon
cd carbon

# 配置环境变量
cp deployment/.env.prod.example .env
nano .env

# 修改以下配置:
# SUPABASE_URL=https://api.yourcompany.com
# SUPABASE_ANON_KEY=从 supabase/.env 复制
# SUPABASE_SERVICE_ROLE_KEY=从 supabase/.env 复制
# SUPER_ADMIN_EMAIL=admin@yourcompany.com
# SUPER_ADMIN_PASSWORD=强密码

# 安装依赖
pnpm install

# 构建应用
pnpm build

# 运行数据库迁移
pnpm db:migrate

# 启动应用
pm2 start deployment/pm2/ecosystem.config.js
pm2 save
pm2 startup
```

### 第 5 步：配置 Nginx 和 SSL（10 分钟）

```bash
# 配置 Nginx
sudo cp deployment/nginx/sites/*.conf /etc/nginx/sites-available/
sudo ln -s /etc/nginx/sites-available/*.conf /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# 安装 SSL 证书
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx \
  -d erp.yourcompany.com \
  -d mes.yourcompany.com \
  -d api.yourcompany.com
```
