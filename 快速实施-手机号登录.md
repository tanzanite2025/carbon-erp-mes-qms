# 快速实施：手机号登录

## 🎯 目标

让 Carbon ERP 支持手机号登录，适配中国市场。

---

## ⚡ 最小可行方案（MVP）

### 方案：手机号 + 工号 + 密码

**优点**：
- ✅ 无需短信服务商（零成本）
- ✅ 实施简单（1-2天）
- ✅ 完全离线可用
- ✅ 适合内部系统

**缺点**：
- ⚠️ 需要管理密码
- ⚠️ 安全性依赖密码强度

#### 3. 修改认证逻辑（1小时）

`packages/auth/src/services/auth.server.ts`：

```typescript
export async function authenticateByPhone(
  phone: string,
  password: string,
  employeeNumber?: string
) {
  const serviceRole = getCarbonServiceRole();
  
  // 查找用户
  let query = serviceRole
    .from("user")
    .select("*")
    .eq("phone", phone);
  
  if (employeeNumber) {
    query = query.eq("employeeNumber", employeeNumber);
  }
  
  const user = await query.maybeSingle();
  
  if (!user.data) {
    return { error: "用户不存在" };
  }
  
  // 验证密码（使用 Supabase Auth）
  const { data, error } = await serviceRole.auth.signInWithPassword({
    email: user.data.email || `${phone}@phone.internal`,
    password,
  });
  
  if (error) {
    return { error: "密码错误" };
  }
  
  return { data };
}
```

## 🚀 完整方案：手机号 + 短信验证码

### 前置条件

1. 注册阿里云账号
2. 开通短信服务
3. 创建短信签名和模板
4. 获取 AccessKey

### 实施步骤

#### 1. 安装依赖（5分钟）

```bash
pnpm add @alicloud/dysmsapi20170525 @alicloud/openapi-client
```

#### 2. 配置环境变量（5分钟）

`.env`：

```bash
# 阿里云短信配置
ALIYUN_SMS_ACCESS_KEY_ID="your_access_key_id"
ALIYUN_SMS_ACCESS_KEY_SECRET="your_access_key_secret"
ALIYUN_SMS_SIGN_NAME="您的公司名"
ALIYUN_SMS_TEMPLATE_CODE="SMS_123456789"
```

#### 3. 创建短信服务（30分钟）

`packages/lib/src/aliyun-sms.server.ts`：

```typescript
import Dysmsapi from '@alicloud/dysmsapi20170525';
import * as $OpenApi from '@alicloud/openapi-client';

const config = new $OpenApi.Config({
  accessKeyId: process.env.ALIYUN_SMS_ACCESS_KEY_ID,
  accessKeySecret: process.env.ALIYUN_SMS_ACCESS_KEY_SECRET,
  endpoint: 'dysmsapi.aliyuncs.com',
});

const client = new Dysmsapi(config);

export async function sendOTP(phone: string, code: string) {
  try {
    const response = await client.sendSms({
      phoneNumbers: phone,
      signName: process.env.ALIYUN_SMS_SIGN_NAME,
      templateCode: process.env.ALIYUN_SMS_TEMPLATE_CODE,
      templateParam: JSON.stringify({ code }),
    });
    
    return { success: true, data: response };
  } catch (error) {
    console.error('发送短信失败:', error);
    return { success: false, error };
  }
}
```

#### 4. 创建 OTP 管理（1小时）

`packages/auth/src/services/otp.server.ts`：

```typescript
import { redis } from "@carbon/kv";
import { sendOTP } from "@carbon/lib/aliyun-sms.server";

const OTP_EXPIRY = 5 * 60; // 5分钟
const RATE_LIMIT = 60; // 1分钟

export async function sendPhoneOTP(phone: string) {
  // 检查频率限制
  const rateLimitKey = `otp:ratelimit:${phone}`;
  const lastSent = await redis.get(rateLimitKey);
  
  if (lastSent) {
    return { 
      success: false, 
      error: "请稍后再试" 
    };
  }
  
  // 生成6位验证码
  const code = Math.floor(100000 + Math.random() * 900000).toString();
  
  // 存储到 Redis
  const otpKey = `otp:${phone}`;
  await redis.setex(otpKey, OTP_EXPIRY, code);
  await redis.setex(rateLimitKey, RATE_LIMIT, "1");
  
  // 发送短信
  const result = await sendOTP(phone, code);
  
  return result;
}

export async function verifyPhoneOTP(phone: string, code: string) {
  const otpKey = `otp:${phone}`;
  const storedCode = await redis.get(otpKey);
  
  if (!storedCode) {
    return { success: false, error: "验证码已过期" };
  }
  
  if (storedCode !== code) {
    return { success: false, error: "验证码错误" };
  }
  
  // 验证成功，删除验证码
  await redis.del(otpKey);
  
  return { success: true };
}
```

#### 5. 修改登录页面（1小时）

`apps/erp/app/routes/_public+/login-phone.tsx`：

```typescript
export default function PhoneLogin() {
  const [step, setStep] = useState<"phone" | "otp">("phone");
  const [phone, setPhone] = useState("");
  const [countdown, setCountdown] = useState(0);

  const sendOTP = async () => {
    const response = await fetch("/api/auth/send-otp", {
      method: "POST",
      body: JSON.stringify({ phone }),
    });
    
    if (response.ok) {
      setStep("otp");
      setCountdown(60);
    }
  };

  return (
    <VStack spacing={4}>
      {step === "phone" ? (
        <>
          <PhoneInput
            name="phone"
            label="手机号"
            value={phone}
            onChange={setPhone}
            defaultCountry="CN"
          />
          <Button onClick={sendOTP}>
            发送验证码
          </Button>
        </>
      ) : (
        <>
          <Input
            name="otp"
            label="验证码"
            placeholder="请输入6位验证码"
            maxLength={6}
          />
          <Button type="submit">
            登录
          </Button>
          <Button 
            variant="ghost" 
            onClick={sendOTP}
            disabled={countdown > 0}
          >
            {countdown > 0 ? `${countdown}秒后重发` : "重新发送"}
          </Button>
        </>
      )}
    </VStack>
  );
}
```

```typescript
export async function action({ request }: ActionFunctionArgs) {
  const { phone, code } = await request.json();
  
  const result = await verifyPhoneOTP(phone, code);
  
  if (result.success) {
    // 查找或创建用户
    const user = await getUserByPhone(phone);
    
    // 创建会话
    const session = await createSession(user.id);
    
    return json({ success: true, session });
  }
  
  return json(result);
}
```
