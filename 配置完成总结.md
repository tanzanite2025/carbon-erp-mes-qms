# ✅ Carbon 项目配置总结

## 🎯 已完成的配置

### 端口修改（避免冲突）

| 服务 | 原端口 | 新端口 | 修改文件 |
|-----|--------|--------|---------|
| Redis | 6379 | **6380** | `docker-compose.yml`, `packages/dev/src/worktree.ts` |
| ERP | 3000 | **4000** | `packages/dev/src/commands/up.ts` |
| MES | 3001 | **4001** | `packages/dev/src/commands/up.ts` |

### 环境变量（.env）

```bash
DEFAULT_LANGUAGE="zh"               # 中文界面
SHARED_REDIS_PORT="6380"            # Redis 端口
DISABLE_RESEND="true"               # 禁用 Resend
AUTH_PROVIDERS="email"              # 邮箱登录
```

---

## 🚀 启动项目

```bash
# 在 Git Bash 中运行
./dev.sh
```

启动后访问：
- **ERP**: http://localhost:4000
- **MES**: http://localhost:4001
- **邮件测试**: https://mail.dev
- **数据库管理**: https://studio.dev

---

## 🔧 常用命令

```bash
# 开发
pnpm dev              # 启动 ERP + MES
crbn status           # 查看状态
crbn down             # 停止服务
crbn reset            # 重置数据

# 数据库
pnpm db:migrate       # 运行迁移
pnpm db:types         # 生成类型
```

---

## 🐛 常见问题

### Docker 连接失败
```bash
docker ps  # 确保 Docker 正在运行
```

### Redis 连接失败
```bash
docker ps | grep redis
# 应该看到：carbon-redis 运行在 0.0.0.0:6380->6379/tcp
```

### 重置环境
```bash
crbn down
crbn reset
crbn up
```
