## 常用命令速查

### SSH连接
```powershell
# 连接服务器
ssh root@服务器IP

# 使用密钥连接（更安全）
ssh -i C:\Users\你的用户名\.ssh\id_rsa root@服务器IP

# 断开连接
exit
```

### 文件传输（使用SCP）
```powershell
# 上传文件到服务器
scp C:\local\file.txt root@服务器IP:/opt/carbon/

# 从服务器下载文件
scp root@服务器IP:/opt/carbon/backup.sql C:\backup\

# 上传整个文件夹
scp -r C:\local\folder root@服务器IP:/opt/carbon/
```

### 查看服务器文件（使用SFTP）
```powershell
# 连接SFTP
sftp root@服务器IP

# SFTP命令
ls              # 列出服务器文件
cd /opt/carbon  # 切换目录
get file.txt    # 下载文件
put file.txt    # 上传文件
exit            # 退出
```

## 故障排查

### SSH连接失败

**问题1: Connection refused**
```
原因: 服务器SSH服务未启动或端口未开放
解决: 
  1. 检查阿里云安全组是否开放22端口
  2. 联系阿里云客服
```

**问题2: Permission denied**
```
原因: 密码错误或密钥不匹配
解决:
  1. 确认密码正确
  2. 在阿里云控制台重置密码
```

**问题3: Connection timeout**
```
原因: 网络问题或IP错误
解决:
  1. 检查服务器IP是否正确
  2. 检查本地网络连接
  3. 尝试ping服务器IP
```

### 文件上传失败

**使用WinSCP:**
1. 打开WinSCP
2. 新建站点
3. 主机名: 服务器IP
4. 用户名: root
5. 密码: 服务器密码
6. 点击"登录"
7. 拖拽文件上传

## 安全建议

### 1. 使用SSH密钥登录（推荐）

**生成SSH密钥:**
```powershell
# 在本地PowerShell执行
ssh-keygen -t rsa -b 4096

# 按提示操作，密钥保存在:
# C:\Users\你的用户名\.ssh\id_rsa
# C:\Users\你的用户名\.ssh\id_rsa.pub
```

**上传公钥到服务器:**
```powershell
# 方式1: 使用ssh-copy-id（如果有）
ssh-copy-id root@服务器IP

# 方式2: 手动复制
# 1. 查看公钥内容
type C:\Users\你的用户名\.ssh\id_rsa.pub

# 2. SSH连接到服务器
ssh root@服务器IP

# 3. 在服务器上执行
mkdir -p ~/.ssh
nano ~/.ssh/authorized_keys
# 粘贴公钥内容，保存

# 4. 设置权限
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
```

**使用密钥登录:**
```powershell
ssh -i C:\Users\你的用户名\.ssh\id_rsa root@服务器IP
```

### 2. 保存SSH配置

**创建SSH配置文件:**
```powershell
# 创建配置文件
notepad C:\Users\你的用户名\.ssh\config
```

**配置内容:**
```
Host carbon-server
    HostName 你的服务器IP
    User root
    Port 22
    IdentityFile C:\Users\你的用户名\.ssh\id_rsa
```

**使用别名连接:**
```powershell
ssh carbon-server
```
