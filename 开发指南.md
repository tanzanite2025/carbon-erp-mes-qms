# Carbon 开发指南

## 🚀 快速启动

```bash
# 在 Git Bash 中运行
./dev.sh
```

启动后访问：
- **ERP**: http://localhost:4000
- **MES**: http://localhost:4001
- **邮件测试**: https://mail.dev
- **数据库管理**: https://studio.dev

### 首次登录

**方式 1: 使用开发绕过邮箱（推荐）**
- 邮箱: `admin@example.com`
- 输入后直接登录，无需密码和验证

**方式 2: 注册新账号**
- 使用任意邮箱注册
- 在 https://mail.dev 查看验证邮件
- 点击验证链接完成注册

## ⚙️ 配置说明

### 端口配置（已修改）

| 服务 | 端口 | 说明 |
|-----|------|------|
| Redis | 6380 | 避免与现有容器冲突 |
| ERP | 4000 | 避免与其他开发环境冲突 |
| MES | 4001 | 避免与其他开发环境冲突 |

### 环境变量（.env）

```bash
DEFAULT_LANGUAGE="zh"               # 中文界面
SHARED_REDIS_PORT="6380"            # Redis 端口
DISABLE_RESEND="true"               # 禁用 Resend
AUTH_PROVIDERS="email"              # 邮箱登录
```

## 🔧 常用命令

```bash
# 开发
pnpm dev              # 启动 ERP + MES
pnpm dev:academy      # 启动 Academy
pnpm dev:starter      # 启动 Starter

# crbn CLI
crbn status           # 查看状态
crbn up               # 启动 Docker
crbn down             # 停止 Docker
crbn reset            # 重置数据

# 数据库
pnpm db:migrate       # 运行迁移
pnpm db:types         # 生成类型

# 翻译
pnpm lingui extract   # 提取翻译
pnpm lingui compile   # 编译翻译
```

## 🐛 常见问题

### Docker 连接失败
```bash
# 确保 Docker Desktop 正在运行
docker ps
```

### Redis 连接失败
```bash
# 检查 Redis 容器
docker ps | grep redis
# 应该看到：carbon-redis 运行在 0.0.0.0:6380->6379/tcp
```

### 重置环境
```bash
crbn down
crbn reset
crbn up
pnpm dev
```

## 💡 提示

- 必须在 **Git Bash** 中运行 `./dev.sh`
- 首次启动需要 5-10 分钟（下载镜像、初始化数据库）
- 邮件不会发送到真实邮箱，在 https://mail.dev 查看
- 默认显示中文界面
