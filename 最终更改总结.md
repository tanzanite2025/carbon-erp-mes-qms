### 3. 添加侧边栏固定展开功能（完整版）

#### 3.1 更新 UI Store（持久化）

**文件**: `apps/erp/app/stores/ui.ts`

**新增功能**:
```typescript
interface UIStore {
  // ... 现有字段
  isSidebarPinned: boolean;           // 新增：固定状态
  setSidebarPinned: (pinned: boolean) => void;  // 新增
  toggleSidebarPin: () => void;       // 新增
}
```

**持久化配置**:
```typescript
persist(
  (set) => ({ ... }),
  {
    name: "carbon-ui-storage",
    storage: createJSONStorage(() => localStorage),
    partializeState: (state) => ({
      isSidebarPinned: state.isSidebarPinned  // 只持久化固定状态
    })
  }
)
```

**优点**:
- ✅ 状态持久化到 localStorage
- ✅ 页面刷新后状态保持
- ✅ 只持久化必要的状态（不持久化搜索模态框）

#### 3.2 更新 PrimaryNavigation

**文件**: `apps/erp/app/components/Layout/Navigation/PrimaryNavigation.tsx`

**新增功能**:
1. **切换按钮**
   - 图标：固定时显示 `LuChevronLeft`（左箭头），未固定时显示 `LuChevronRight`（右箭头）
   - 位置：侧边栏顶部，与其他导航项样式完全一致
   - 文字提示：展开时显示"收起"或"展开"
   - 整个按钮区域（图标+文字）都可点击

2. **移动端适配**
   - 移动端不显示切换按钮
   - 移动端强制不固定（`effectiveIsPinned = false`）

3. **交互方式**
   - ✅ **仅支持点击切换**：点击按钮展开/收起侧边栏
   - ❌ **移除 hover 功能**：不再支持鼠标悬停自动展开（避免误操作）
   - 编辑模式时自动展开

**代码逻辑**:
```typescript
const isMobile = useIsMobile();
const isPinned = useUIStore((state) => state.isSidebarPinned);
const setSidebarPinned = useUIStore((state) => state.setSidebarPinned);

// 移动端强制不固定
const effectiveIsPinned = isMobile ? false : isPinned;

// 展开状态：固定展开 或 编辑模式
const isOpen = effectiveIsPinned || editMode.isEditing;

// 点击切换：直接设置相反的状态
const handleTogglePin = () => {
  setSidebarPinned(!effectiveIsPinned);
};
```

**重要改进**:
- ✅ 移除了所有 `onMouseEnter`/`onMouseLeave` hover 逻辑
- ✅ 使用 `setSidebarPinned(!effectiveIsPinned)` 而不是 `togglePin()` 避免双击 bug
- ✅ 按钮样式与 `NavigationIconLink` 完全一致
- ✅ 图标定位：`absolute left-3 top-3`
- ✅ 文字定位：`absolute left-7 group-data-[state=expanded]:left-12`

**状态**: ✅ 已完成，功能完整

## 🎓 学习要点

### Zustand Persist
```typescript
// 正确的持久化配置
persist(
  (set) => ({ ... }),
  {
    name: "storage-key",
    storage: createJSONStorage(() => localStorage),
    partializeState: (state) => ({
      // 只持久化需要的字段
      field1: state.field1
    })
  }
)
```

### 移动端适配
```typescript
const isMobile = useIsMobile();
const effectiveValue = isMobile ? mobileValue : desktopValue;
```

### 条件 Hover（已移除）
```typescript
// ❌ 旧方案：条件 hover（容易误触发）
onMouseEnter={condition ? undefined : handler}
onMouseLeave={condition ? undefined : handler}

// ✅ 新方案：完全移除 hover，只用点击
// 无 hover 相关代码
```

### 避免双击 Bug
```typescript
// ❌ 错误：使用 toggle 函数可能导致双击 bug
const handleToggle = () => {
  toggleSidebarPin(); // React 状态更新可能不同步
};

// ✅ 正确：直接设置相反的状态
const handleToggle = () => {
  setSidebarPinned(!effectiveIsPinned); // 基于当前值设置
};
```
